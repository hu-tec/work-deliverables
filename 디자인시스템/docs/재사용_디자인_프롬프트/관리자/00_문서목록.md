# 관리자 재사용 디자인 프롬프트

BO 관리자 화면, 대량 테이블, 검색조건, inline Drawer를 만들 때 쓰는 최종 기준 문서다.

## 기준 문서

| 문서 | 용도 |
|---|---|
| `03_BO_관리자화면_기준.md` | 관리자 화면 전체 shell, 업무 전환, 액션, 상세 편집 기준 |
| `04_BO_관리자용_레이아웃_기준.md` | 4단 그리드, 고밀도 업무 패널, 대시보드/상세 배치 기준 |
| `02_BO_테이블레이아웃_기준.md` | 대량 테이블, 검색필터, inline Drawer 테이블 화면 기준 |

## 컴포넌트 문서

| 문서 | 용도 |
|---|---|
| `components/검색필터_기준.md` | 고밀도 검색조건과 필터 그룹 구성 기준 |
| `components/상태_유형_칩_기준.md` | 상태값, 유형값, 구독/등급/결제수단 표시 기준 |
| `components/인라인_Drawer_기준.md` | 행 클릭 상세, 우측 inline Drawer, 넓게보기 작업영역 기준 |
| `components/테이블선택_Drawer_상세패널_기준.md` | 테이블 행 선택 후 열리는 상세 작업 패널, 상태 UI, 고정/변동 정보 기준 |

## 사용 순서

1. 관리자 화면 전체를 만들 때는 `03_BO_관리자화면_기준.md`를 먼저 적용한다.
2. 대시보드나 상세 작업 화면이면 `04_BO_관리자용_레이아웃_기준.md`를 함께 적용한다.
3. 대량 목록/검색/상세 테이블이면 `02_BO_테이블레이아웃_기준.md`를 함께 적용한다.
4. 상태/유형/필터/Drawer가 있으면 `components` 문서를 같이 적용한다.
5. 테이블 행 선택 후 상담/시험/결제/정산을 처리하는 상세 패널은 `components/테이블선택_Drawer_상세패널_기준.md`를 우선 적용한다.
