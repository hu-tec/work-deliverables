사용 순서

  1. 관리자 화면 전체를 만들 때는 03_BO_관리자화면_기준.md를 먼저 적용한다.
  2. 대시보드나 상세 작업 화면이면 04_BO_관리자용_레이아웃_기준.md를 함께 적용한다.
  3. 대량 목록/검색/상세 테이블이면 02_BO_테이블레이아웃_기준.md를 함께 적용한다.
  4. 상태/유형/필터/Drawer가 있으면 components 문서를 같이 적용한다.
  5. 테이블 행 선택 후 상담/시험/결제/정산을 처리하는 상세 패널은 components/테이블선택_Drawer_상세패널_기준.md를 우선 적용한다.
