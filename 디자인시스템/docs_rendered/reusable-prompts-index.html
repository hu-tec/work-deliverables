사용 순서

  1. 사용자 화면을 만들 때는 사용자/01_사용자화면_디자인시스템_기준.md를 먼저 적용한다.
  2. 관리자 화면 전체를 만들 때는 관리자/03_BO_관리자화면_기준.md를 먼저 적용한다.
  3. 관리자 대시보드나 상세 작업 레이아웃은 관리자/04_BO_관리자용_레이아웃_기준.md를 함께 적용한다.
  4. 관리자 대량 테이블 화면을 만들 때는 관리자/02_BO_테이블레이아웃_기준.md를 함께 적용한다.
  5. 상태/유형/필터/Drawer가 필요한 관리자 화면은 관리자/components 문서를 함께 적용한다.
  6. 테이블 행 선택 후 상세 작업을 처리하는 화면은 관리자/components/테이블선택_Drawer_상세패널_기준.md를 함께 적용한다.
  7. 새 프로젝트에 복사할 때는 이 폴더 전체를 기준 프롬프트로 전달한다.

공통 금지

  • 기준 파일에 없는 새로운 색상 팔레트를 임의로 만들지 않는다.
  • 큰 카드, 과한 radius, 넓은 여백, 마케팅식 장식을 BO 테이블 화면에 적용하지 않는다.
  • 상태값을 색상 텍스트만으로 표시하지 않는다.
  • 날짜와 카운트 표기 형식을 화면마다 다르게 쓰지 않는다.
  • 사용자 화면과 관리자 화면의 밀도, 글자 크기, 정보 구조를 섞지 않는다.
