# 용어 사전

| 용어 | 의미 | 사용 기준 |
|---|---|---|
| Root | 워크스페이스 최상위 문서 영역 | 전역 규칙과 공통 정책을 둘 때 사용 |
| Local | 하위 폴더별 문서 영역 | 특정 사이트, 패키지, 도메인 규칙을 둘 때 사용 |
| SSOT | Single Source of Truth, 단일 진실 공급원 | 중복 없이 하나의 기준 문서를 유지할 때 사용 |
| Override | 상위 규칙을 허용된 범위에서 재정의하는 것 | 로컬 규칙이 전역 규칙과 다를 때 사용 |
| Context | 현재 상태와 작업 배경 | 누적 기록이 아니라 최신 상태 중심 |
| Handoff | 세션 인계 기록 | 다음 작업자가 바로 이어서 작업할 수 있게 작성 |
| ADR | Architecture Decision Record | 중요한 설계 결정을 기록하는 문서 |
