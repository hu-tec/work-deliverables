# HutechC Exam 사이트 비교

## 요약

- `https://hutechc-exam.vercel.app/ko`는 시험 운영 포털입니다. 시험 찾기, 시험장 입장, 접수 상태 확인, 응시 준비 확인처럼 사용자가 바로 처리해야 하는 업무를 우선합니다.
- `https://hutechc-exam-vibe2.vercel.app/ko`는 어두운 톤의 제품 소개형 랜딩 페이지입니다. 기존 수험자 업무보다는 플랫폼 개념, 서비스 흐름, 평가 모델, MVP 우선순위를 설명하는 데 집중합니다.
- 이번 클론코딩은 첫 번째 사이트를 기준으로 했습니다. 실제 포털 UX와 업무 중심 정보 구조가 더 명확하기 때문입니다.

## hutechc-exam.vercel.app 특징

- 시각 톤: 밝은 배경, 흰색 패널, 파란색 주요 버튼, 촘촘한 대시보드 밀도.
- 헤더: 60px 고정 바에 주요 사용자 메뉴, 역할별 진입점, 로그인, 마이페이지, 알림, 장바구니, 프로필을 배치.
- 첫 화면: breadcrumb, 시험 검색 입력, 시험장 입장 CTA, 오늘 입장 가능한 시험 카드.
- 본문: 오늘 지표, 빠른 메뉴, 접수 진행 단계, 검색 필터, 접수 내역 테이블, 응시 준비 체크리스트, 시험 전 안내.
- 제품 성격: 마케팅 사이트보다 로그인 후 사용하는 포털/대시보드에 가깝습니다.

## hutechc-exam-vibe2.vercel.app 특징

- 시각 톤: 어두운 배경, radial gradient, 글래스 패널, 큰 라운드, 시안/바이올렛 강조색.
- 헤더: 제품 브랜딩과 일부 핵심 링크만 둔 축약형 구조.
- 첫 화면: 플랫폼 소개 문구, 운영 상태 요약, “시험 접수 시작하기” CTA.
- 본문: 서비스 단계 카드, 평가 기준표 progress bar, 시험 카탈로그, 핵심 기능 카드, 서비스/가격 카드, MVP 우선순위.
- 제품 성격: 수험자 업무 처리보다 고객이나 이해관계자에게 플랫폼을 소개하는 랜딩 페이지에 가깝습니다.

## 실무적 차이

| 비교 항목 | hutechc-exam | hutechc-exam-vibe2 |
| --- | --- | --- |
| 주요 목적 | 기존 사용자의 업무 처리 | 제품 설명과 전환 유도 |
| 화면 밀도 | 높음, 테이블/대시보드 중심 | 중간, 카드/랜딩 중심 |
| 내비게이션 | 전체 포털 메뉴 | 축약된 제품 메뉴 |
| CTA 우선순위 | 내 시험 찾기 / 시험장 입장 | 시험 접수 시작 |
| 적합 대상 | 수험자, 운영자, 로그인 사용자 | 잠재 고객, 의사결정자, 평가자 |
| 클론 대상 여부 | 대상 | 비교 참고 대상 |

## 구현 메모

- 클론은 Next.js App Router의 `app/[locale]` 구조를 사용합니다.
- shadcn 스타일 primitive는 `components/ui`에 로컬로 두었습니다. 추후 shadcn CLI 도입이나 컴포넌트 확장이 쉽습니다.
- 테마 값은 `app/globals.css`의 CSS 변수로 관리합니다. `data-theme` 또는 토큰 값을 바꾸면 컴포넌트 수정 없이 색상 체계를 바꿀 수 있습니다.
- 다국어 사전은 `lib/i18n.ts`에 있습니다. 현재 한국어/영어가 연결되어 있고, `locales`와 `dictionaries`를 확장하면 언어를 추가할 수 있습니다.
