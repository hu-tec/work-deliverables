# AITP 통합 시험 플랫폼 구현본

## 실행

브라우저에서 아래 파일을 직접 열면 됩니다.

- `public.html`: 공개 홈페이지
- `login.html`: 로그인/계정 진입
- `index.html`: 로그인 후 역할별 내부 운영 앱

공개 페이지에서 로그인 페이지로 이동하고, 로그인 페이지에서 선택한 권한에 맞는 내부 운영 앱 홈으로 진입합니다.

## 구현 범위

- 공개 홈페이지
- 인증/계정 및 권한별 홈 이동
- 공개 홈페이지와 로그인 페이지 독립 분리
- 수험생 시험 접수, 결제, 응시, 결과, 자격증 신청 플로우
- 출제자 번역형/프롬프트형/단답형 문제 작성 플로우
- 채점자 채점, 평가서, 점수/코멘트 저장 플로우
- 관리자 회원, 그룹, 관리자, 전문가, 시험, 문제, 채점, 결제, 정산, 환불, 코드, 카테고리, 문의, 통계 관리
- 2025 양식관리, 템플릿, 창작작업실, 메타-T 번역, 번역/편집, 장바구니, 결제
- vibecoding 대량관리 탭, 다중 체크박스 필터, 가로 스크롤 테이블, 엑셀/파일/SMS/메일 액션
- `DB_정리` 문서와 CSV 기준 샘플 더미데이터

## 검증

- `node --check app.js` 통과
- `plan.md` 검증 체크리스트 완료 표시
