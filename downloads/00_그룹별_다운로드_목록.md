# 그룹별 다운로드 ZIP 목록

## index.html 연결 예시

`index.html`에서 각 그룹 제목 옆에 다운로드 링크를 붙일 때는 아래 경로를 사용한다.

```js
const DOWNLOADS = {
  '디자인시스템/재사용 UI': 'downloads/디자인시스템_재사용_UI.zip',
  '관리자/BO 업무화면': 'downloads/관리자_BO_업무화면.zip',
  '사용자/랜딩·신청 화면': 'downloads/사용자_랜딩_신청_화면.zip',
  '내부 업무도구/자동화': 'downloads/내부_업무도구_자동화.zip',
  '기획·전략 자료': 'downloads/기획_전략_자료.zip',
  '교육·시험·콘텐츠 자료': 'downloads/교육_시험_콘텐츠_자료.zip',
  '개발·운영 매뉴얼': 'downloads/개발_운영_매뉴얼.zip',
  '보관/원본 자료': 'downloads/보관_원본_자료.zip'
};
```

분석 자료는 `index.html`에 묶지 않고 `analyze.html`에서 별도로 본다.

```js
const ANALYZE_DOWNLOAD = 'downloads/분석.zip';
```

HTML 예시:

```html
<a class="download-link" href="downloads/디자인시스템_재사용_UI.zip" download>다운로드</a>
```

## 포함 기준

- `.html`, `.md`, `.css`, `.js`, `.json`, `.csv`, 이미지, `.docx`, `.txt` 중심으로 포함했다.
- `.git`, `.codex`, `.agents`, `node_modules`, `downloads` 폴더는 제외했다.
- HTML 화면은 단독 파일만 받으면 CSS/JS/이미지가 빠질 수 있으므로, 실제 참고에 필요한 인접 리소스도 함께 포함했다.
