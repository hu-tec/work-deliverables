# TESOL-v3 `/courses/tesol` 분석

분석일: 2026-05-22  
대상: https://hu-tec.github.io/TESOL-v3/courses/tesol

## 1. 사이트 성격

React 기반 SPA로 제작된 신규 배포 버전이다. 실제 HTML은 `root`만 가진 앱 셸이고, 콘텐츠는 번들 JS에서 렌더링된다. 라우팅 기준 basename은 `/TESOL-v3/`이며, 주요 경로는 홈, 교육원 소개, 입학안내, 과목 소개, 커뮤니티, 문의, 로그인, 회원가입, 신청서다.

브랜드는 `A. TESOL`이며, 부제는 `국제 영어교사 양성과정`이다. 기존 사이트보다 TESOL 중심성이 강하고, UI/UX와 전환 흐름이 가장 현대적이다.

## 3. `/courses/tesol` 콘텐츠 구조

TESOL 페이지는 과정 상세 랜딩에 가깝다. 핵심 문구는 다음과 같다.

- `8주 완성 · 120시간 몰입`
- `전세계 학생과 함께하는 100% 영어 수업`
- `세계 어디서나 통하는 자격증`
- `발표·수업 관리 실습`
- `국제 TESOL 자격증`
- `영어 수업·발표 자신감`
- `국내 교사·강사 취업`
- `글로벌 교사 커리어`
- `비즈니스 활용 능력`

과정 상세:

| 항목 | 내용 |
|---|---|
| 과정명 | TESOL 교육 과정 |
| 상태 | 모집중 |
| 과정구분 | 정규과정 |
| 커리큘럼 | 국제 TESOL 커리큘럼 |
| 교육기간 | 8주 완성, 120시간 |
| 수업방식 | 100% 영어 원어민 집중 수업 |
| 자격인증 | 수료증 및 국제 자격증 발급 |

설명 문구는 TESOL을 `영어를 모국어로 하지 않는 사람들에게 영어를 가르치는 국제 공인 영어교사 양성 과정`으로 소개하고, 이론보다 원어민 수업과 교실 실습 중심 교육을 강조한다.

## 4. 홈/입학/커뮤니티 연계

홈은 `글로벌 시대, 영어로 가르치는 전문가를 양성합니다`를 메인 메시지로 사용한다. 성과 지표는 교육경험 23년+, TESOL 졸업생 30,000+, 글로벌 네트워크 12개국+, 전문과정 6개 분야로 구성된다.

입학안내에는 다음 요소가 있다.

- 지원서 작성
- 전형료 입금
- 레벨테스트
- 합격자 발표
- 등록·납부
- 서류 제출
- 수업료: 과정별 1,800,000원 ~ 2,800,000원대
- 수업운영: 120시간 / 8주 집중 과정
- 지역 일정: 서울, 부산, 대구, 대전, 경기도

커뮤니티에는 공지사항, FAQ, 졸업생 후기 구조가 있으며 예시 콘텐츠로 2026년 3월 서울 TESOL 과정 개강 안내, AI 번역 교정 전문가 과정 신규 오픈, 홈페이지 리뉴얼 기념 할인 이벤트 등이 들어 있다.

## 5. 강점

- 최종 통합 사이트의 기본 골격으로 가장 적합하다.
- 모바일 메뉴, 드롭다운, 신청 CTA, 문의 폼, 신청서가 이미 구성되어 있다.
- TESOL 페이지가 전환형 랜딩 구조를 갖고 있어 사용자가 과정 가치와 신청 경로를 빠르게 이해할 수 있다.
- 신청서가 2단계 폼으로 구성되어 있고 신청 유형별 기본값을 분기할 수 있다.
- 기존 사이트보다 카피, 정보 구조, 시각 위계가 명확하다.

## 6. 문제점

- `/courses/tesol` 직접 URL은 HTML 본문이 비어 있고 JS 렌더링에 의존한다. SEO가 중요한 교육 사이트라면 SSR/SSG 또는 정적 메타 태그 보강이 필요하다.
- 연락처가 `02-6207-9090`, 입학안내의 문의전화는 `02-1234-5678`로 불일치한다.
- 일부 문구와 데이터는 샘플 또는 검증 전 상태로 보인다. 예: 수업료 범위, 전국 일정, FAQ/공지 예시.
- Footer의 레벨테스트/설명회/자격증 링크 중 일부는 `href="#"`로 남아 있어 실제 라우팅 연결이 필요하다.
- 회원가입 약관에 깨진 문자 `���지`가 포함되어 있다.
- `A. TESOL` 브랜딩은 명확하지만 기존 `타임스테솔`, `타임스미디어`와의 관계 설명이 첫 화면에서 약할 수 있다.
