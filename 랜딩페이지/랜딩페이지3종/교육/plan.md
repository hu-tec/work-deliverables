# 시험 랜딩 홈페이지 레이아웃 정의

## 구현목표
- `/site-template`에 있는 페이지들을 기준으로 템플릿을 생성/수정

전체적으로 글자를 -2px줄이기.

## 교육과정 상세
과정 상세내용은 2단구조
커뮤니티 : `/community-redesign/design-a-modern`을 채택
수강료(우측패널) : `/community-redesign/design-b-modern`을 채택


## 교육과정 상세 레이아웃 구조
과정소개는
- 이미지 | 과정내용 : 2단구조 (repeat(2, 1fr))
- 이과정필요성 | 추천 : 2단구조 (repeat(2, 1fr))
- 교육전후 비교 : 1줄 
- 과정특징 : 1줄
- 강점 한줄(4단) > 없는경우, ai로 생성 : 4단구조 (repeat(4, 1fr))
- 교육과정이 2개이상인경우, 최소 2단 ~ 4단을 최대

## 헤더
- 시험 소개
- 수강신청
- 시험 안내
- 커뮤니티
- 1:1문의
- 헤더 우측 액션은 [신청하기]만 유지하고, 기존 [교육원 소개] 버튼은 삭제

로그인 회원가입 우측으로

## 수강신청
- 기존 `자격증 소개` 메뉴/페이지는 `수강신청`으로 변경
- 수강신청 강좌 조회에서 `수강신청을 진행할 센터`, `구분`, `분야/전공` 각각에 `전체` 선택지를 제공
- `전체` 선택 시 해당 조건은 필터링하지 않고 전체 강좌 대상으로 조회

## 산출
- 과정에 대한 설명 md를 산출
