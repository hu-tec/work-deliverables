# 교육과정 설명 (layout_redesign 산출물)

> plan.md "산출 — 과정에 대한 설명 md를 산출" 항목.
> 기준 데이터: `layout_redesign/data.js` (기존 데이터 축약 없이 반영).
> 상세 페이지: `course-detail.html?id=<id>`

## 과정 상세 레이아웃 (공통)

과정 상세는 **2단 구조**(본문 + 우측 수강료 패널)이며, 본문은 아래 순서로 구성됩니다.

1. **과정소개** — 이미지 | 과정내용 (2단, `repeat(2,1fr)`)
2. **과목 소개** — 과목/모듈 카드 그리드 _(참고: TESOL-v3 ITT 과목소개)_
3. **필요성 | 추천** — 왜 필요한가 | 추천 대상 (2단, `repeat(2,1fr)`)
4. **교육 전·후 비교** — 1줄 (전/후 2열 표)
5. **과정 특징** — 1줄 (데이터 필드에서 파생: 기간·대상·운영방식)
6. **핵심 강점** — 4단 (`repeat(4,1fr)`). 데이터가 4개 미만이면 **AI 추천**으로 보충(`AI 추천` 표기), 2~4단으로 적응
7. **교육과정 안내** — 교육 목표 · 구성 · 운영방식 · 수료 기준 _(참고: ai-teacher 교육과정안내)_
8. **주차별 커리큘럼** — 표
9. **수강안내** — 접수·일정·수강료·납부·환불·증명 + 신청 절차 4단계 _(참고: ai-teacher 수강안내)_
10. **책임 강사** / **FAQ** / **수강후기**

우측 **수강료 패널**(design-b 채택)은 수강료·기간·일시·대상·접수·상태와 신청/상담 버튼을 sticky로 고정합니다.

> 추가 데이터 필드(`data.js`): `subjects[[과목명, 설명]]`(과목 소개), `goals[]`·`completion`(교육과정 안내/수료 기준). 수강안내의 접수·납부·환불 절차는 과정 공통 표준으로 구성됩니다.

## 1. AI 강사 양성과정 `id=ai-teacher` — 모집중 · 실무과정

**한 줄 정의** : AI 자체를 가르치는 과정이 아니라, AI툴과 프롬프트를 활용해 초등 방과후 수업을 더 재미있고 효과적으로 운영하는 교사를 양성하는 과정입니다.

**과정소개**
- 이 과정은 AI 자체를 가르치는 과정이 아니라, AI툴과 프롬프트를 활용해 초등 방과후 수업을 더 재미있고 효과적으로 운영하는 교사를 양성하는 과정입니다.
- 이미지, 영상, 음성, 문서형 AI툴을 활용하여 수업자료를 만들고, 활동형 콘텐츠를 구성하고, 학생 수준과 주제에 맞게 수업을 설계하는 실전 중심 구조로 운영됩니다.
- 특히 방과후 현장에 맞게 영어, 창의활동, 문화, 발표, 프로젝트형 수업으로 연결할 수 있도록 구성됩니다.

**왜 이 과정이 필요한가**
- 기존 방과후 수업은 자료 준비 시간이 오래 걸리고 반복업무가 많기 때문
- 학생 참여를 높이기 위해 이미지·영상·활동형 콘텐츠 구성이 중요하기 때문
- 학부모와 학교가 신뢰할 수 있도록 AI 윤리 이해가 반드시 필요하기 때문
- AI를 활용한 영어·창의형 수업이 학교 차별화 프로그램으로 확장될 수 있기 때문

**핵심 강점**
- AI툴 출력부터 수정·보완까지 직접 다루는 실전형 구성
- 방과후 현장에서 바로 쓸 수 있는 수업자료 제작 중심 학습
- 영어, 문화, 발표, 프로젝트형 수업으로 확장 가능한 구조
- AI 윤리를 함께 익혀 학부모와 학교 신뢰 확보 가능

**이런 분께 추천**
- 초등 방과후 교사로 활동하고 싶은 분
- 영어수업과 AI 활용을 결합하고 싶은 분
- 학생 참여형 수업 콘텐츠를 직접 만들고 싶은 분
- 테솔, 영어교육, 창의교육 경험을 AI와 연결하고 싶은 분
- 학교, 기관, 공부방 등에서 차별화된 수업을 운영하고 싶은 분

**교육 전 · 교육 후**

| 교육 전 | 교육 후 |
|---|---|
| 수업자료 준비에 시간이 오래 걸리고 반복업무가 많음 | AI툴로 수업자료·활동지를 빠르게 제작 |
| 이미지·영상 등 참여형 콘텐츠 구성이 어려움 | 이미지·영상·활동형 콘텐츠로 학생 참여를 높임 |
| AI 활용의 윤리 기준이 불명확 | 저작권·개인정보 등 AI 윤리를 학부모·학교에 설명 가능 |

**주차별 커리큘럼**

| 구분 | 주제 | 주요 내용 |
|---|---|---|
| 1주차 | 오리엔테이션·AI툴 이해 | 이미지·영상·음성·문서형 AI툴 개요와 초등 방과후 활용 사례 |
| 2주차 | 프롬프트·수업 설계 | 학생 수준·주제별 프롬프트 작성과 수업 흐름 설계 |
| 3주차 | 수업자료 제작 | 이미지·영상·활동지 등 방과후 수업자료 직접 제작 |
| 4주차 | 활동형 콘텐츠 구성 | 영어·창의·문화·발표·프로젝트형 활동 콘텐츠 구성 |
| 5주차 | 시연·AI 윤리 | 수업 시연·피드백, 저작권·개인정보 등 AI 윤리 정리 |

**과목 소개**
- 이미지·영상 AI툴 — 이미지·영상 생성 도구로 수업 시각자료와 활동 영상을 제작
- 음성·문서 AI툴 — 음성 합성과 문서 생성으로 듣기 자료·학습지를 제작
- 프롬프트 설계 — 학생 수준·주제에 맞춘 프롬프트 작성
- 활동형 콘텐츠 구성 — 영어·창의·문화·발표·프로젝트형 활동 구성
- AI 윤리 — 저작권·개인정보·검수 기준 이해·적용

**교육과정 안내 (교육 목표 / 수료 기준)**
- 목표: ①AI툴로 방과후 수업자료를 스스로 제작 ②학생 참여형 활동 콘텐츠 직접 설계 ③AI 윤리 기준을 학부모·학교에 설명
- 수료 기준: 총 출석 80% 이상 + 수업자료 포트폴리오 제출 시 수료증 발급

**수강안내** : 온라인 신청 또는 1:1 문의 후 상담→반 배정 · 수강료 상담 후 안내(분할 가능) · 카드/계좌 납부 · 교육 시작 전 100% 환불(시작 후 진행률 적용) · 절차: 상담·신청 → 접수·납부 → 수강 → 수료·발급

**운영 정보** : 교육기간 5주(주 2회, 총 40시간) · 일시 기수별 공지(평일 저녁반/주말반) · 대상 초등 방과후 교사·예비 교사 · 수강료 상담 후 안내(분할 가능)

## 2. TESOL 영어교육 과정 `id=tesol` — 모집중 · 정규과정

**한 줄 정의** : 영어 교수법, 수업 설계, 시연, 학습자 관리까지 실제 교육 현장에서 필요한 역량을 체계적으로 훈련합니다.

**과정소개**
- 영어 교수법, 수업 설계, 시연, 학습자 관리까지 실제 교육 현장에서 필요한 역량을 체계적으로 훈련합니다.
- 수업안 작성과 발표 피드백을 반복하며 강사 포트폴리오로 활용할 결과물을 완성합니다.

**핵심 강점** : 국제 표준 교수법 학습 / 수업 시연·피드백 반복 훈련 / 수료증 발급
**추천 대상** : 영어 강사를 준비하는 예비 강사 / 수업 운영 방식을 체계화하려는 현직 교사

**커리큘럼** : 1~2주차 영어 교수법 기초 / 3~4주차 수업 설계 / 5~6주차 수업 시연 / 7~8주차 평가·포트폴리오
**과목 소개** : 영어 교수법 · 수업 설계 · 수업 시연 · 학습자 관리
**교육과정 안내** : 목표 — 교수법의 수업안 구체화·시연 검증, 학습자 관리·평가 전략 / 수료 — 출석 충족 + 시연·포트폴리오 평가 통과 시 수료증
**수강안내** : 1:1 상담→반 배정 · 수강료 상담 안내 · 카드/계좌 납부 · 시작 전 100% 환불 · 절차 상담→접수·납부→수강→수료
**운영 정보** : 8주(120시간) · 기수별 공지 · 대상 교사·예비 강사

## 데이터/구현 메모

- 4개 과정 모두 `data.js`에 정의(`COURSES`), 공지 7건(`NOTICES`)도 동일 파일에서 관리. 추가·수정은 `data.js`만 변경.
- 과정이 추가될 경우 상세의 **핵심 강점** 칼럼 수는 강점 개수에 따라 2~4단으로 자동 조정(`data-cols`), 4개 미만이면 AI 추천 항목으로 보충됩니다.
- 전체 글자 크기는 기준 대비 -2px(base 13px) 적용.
- 헤더: `시험 소개 · 자격증 소개 · 시험 안내 · 커뮤니티 · 1:1문의` + 우측 `[신청하기][교육원 소개]`, 상단 우측 `로그인 · 회원가입`.
